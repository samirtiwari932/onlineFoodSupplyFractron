# Organic Food Supply Website (MERN)

A full-stack e-commerce application for organic food supply, focusing on sustainability.

## Features
- **User**: Browse products, Cart, Checkout (Stripe), Order History.
- **Seller**: List products, Manage inventory.
- **Admin**: Approve/Reject products, Platform oversight.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Shadcn UI.
- **Backend**: Node.js, Express, MongoDB.
- **Payments**: Stripe.

## Setup
1.  **Server**:
    ```bash
    cd server
    npm install
    npm start
    ```
2.  **Client**:
    ```bash
    cd client
    npm install
    npm run dev
    ```
